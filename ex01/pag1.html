Formatações

Negrito

A tag <strong> significa que o termo delimitado possui força dentro da frase. Logo ele aparecerá em Negrito.

Itálico

Para colocar um termo em itálico use <em> que significa que queremos dar ênfase (empashis).

Marcar Texto

Marque um texto quando quiser dar uma ênfase descomunal utilizando a tag <mark>.

Texto Deletado

Utilize a tag <del> para dizer que um certo trecho não deve ser considerado.

Texto Inserido

Se colocarmos um texto dentro da tag <ins> estamos dizendo que ele está ali e deve ser lido com atenção.

Texto Sobrescrito e Subscrito

A tag <sub> para sobrescrito e para subscrito a tag <sup>.

Trecho de código

Delimite seu código usando a tag <code>.

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é par.') else: print(f'O número {num} é ímpar.')

E para manter a formatação do código você pode usar a tag <pre>.


        num = int(input('Digite um número'))
        if num % 2 == 0:
            print(f'O número {num} é par.')
        else:
            print(f'O número {num} é ímpar.')

Citações

Para criar uma citação você pode usar a tag <q> (quote, que significa citar).O texto vai receber automaticamente aspas, mas nenhum deslocamento.

Podemos criar citações mais longa (em bloco) e que tenham um parágrafo só para si. Nesse caso, utilizamos a tag <blockquote>

Esquecer é uma necessidade. A vida é uma lousa, em que o destino, para escrever um novo caso, precisa de apagar o caso escrito.

Abreviações

Utilize a tag <abbr> em HTML para deixar claro ao usuário o significado de uma certa sigla

Texto Invertido

Com a tag <bdo> (bi-directional override) você pode inverter a posição do texto. Utilize o parâmetro dir para indicar a direção. Tudo ok?
